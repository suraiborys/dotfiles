# DOTFILES

Automated dotfiles setup using Ansible.

## Prerequisites
- Ansible installed (e.g., via pip or your package manager)
- macOS/Linux shell environment

## Available dotfiles
- [x] ghostty
- [x] zed
- [x] nvim
- [X] tmux

## Quick Start
```bash
ansible-playbook -i inventory/hosts playbook.yml
```
